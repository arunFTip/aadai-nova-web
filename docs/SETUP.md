# Frontend Setup

---

# Requirements

- Node.js 20+
- npm

---

# Install Dependencies

```bash
npm install
```

---

# Run Development Server

```bash
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```

---

# Build

```bash
npm run build
```

---

# Environment Variables

## .env

```env
VITE_API_URL=http://127.0.0.1:8000/api
```
